Core Features

Everything You Need,
Nothing You Don't

PlaySense runs 100% locally on your Windows PC. No internet required, no data ever leaves your machine.

🔍

Hardware Scan

Detects your CPU, GPU, RAM, and OS in real time using native Windows APIs. Automatically assigns a performance tier so every optimization is tailored to your exact machine.

📂

Game Import

Import from Steam, Epic, GOG or any .exe folder using the native Windows File Dialog. All paths are saved persistently to a local JSON database.

⚙️

Smart Tweaks

Scans game folders and AppData for config files. Modifies fields like ShadowQuality, VolumetricFog, and ResolutionScale. Includes a dedicated Unreal Engine 5 injector.

🤖

AI Crash Analyzer

Reads Windows Event Viewer logs and game dump files. Matches error codes against known failure patterns and recommends actionable fixes.

How It Works

Three Steps to
Maximum FPS

No configuration files to edit. No registry keys to hunt down. Just launch PlaySense and let it work.

1

Scan Your System

PlaySense detects your CPU, GPU, RAM, and OS using native Windows APIs. It assigns a performance tier that guides every optimization decision.

2

Import Your Games

Auto-scan finds Steam, Epic, and GOG libraries. For everything else, use the native file dialog to import any .exe or game folder.

3

Optimize & Play

One click applies tier-appropriate settings to every config file it finds. If something crashes, the analyzer tells you exactly what to fix.
